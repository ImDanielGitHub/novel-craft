# Code Of Conduct

Novel Craft uses a simple standard: be practical, respectful, and safe to collaborate with.

Do:

- be kind and direct
- assume good intent while still reviewing carefully
- keep critique about the work, not the person
- respect copyright and writer ownership
- make space for beginner and expert writers

Do not:

- harass, threaten, or demean contributors
- submit copyrighted story text you do not have the right to use
- use the project to build scraping, imitation, or plagiarism workflows
- pressure maintainers to merge or release unreviewed code

Maintainers may close issues, remove comments, block users, or reject contributions that make the project unsafe or unmaintainable.
