# Novel Character Review

Use this skill when characters feel generic, passive, emotionally repetitive, or out of voice.

## Workflow

```bash
novel-craft review <draft.md> --rubric character
novel-craft audit repetition <draft.md>
```

Update continuity if the scene changes the character:

```bash
novel-craft memory extract <draft.md> --scene-id "<scene-id>"
```

## Review Questions

- Does the character make a meaningful choice?
- Does motivation appear in behaviour rather than only narration?
- Does the scene move the wound, false belief, want, need, or relationship state?
- Does the character's voice match their matrix?
- Are sensory signatures used specifically, or are generic gestures repeated?
- Is the emotion progressing, escalating, reversing, or forcing action?

## Rules

- A character sheet is live state, not a static bio.
- Track surface, sensory signature, psychology, behaviour, relationships, continuity, knowledge, current state, and do-not-overuse fields.
