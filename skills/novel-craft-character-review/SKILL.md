# Novel Craft Character Review

```bash
novel-craft review <draft.md> --rubric character
novel-craft audit repetition <draft.md>
novel-craft eval chapter <draft.md> --json
```

Check whether each major character has a visible want, private pressure, fear, wound, contradiction, relationship leverage, and a meaningful choice. Update continuity only when a draft has been accepted.
