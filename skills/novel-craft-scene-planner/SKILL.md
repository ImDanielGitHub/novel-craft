# Novel Craft Scene Planner

```bash
novel-craft scene create "<scene-id>" --pov "<pov>" --location "<place>" --goal "<want>" --conflict "<resistance>" --turn "<change>" --stakes "<cost>"
novel-craft matrix build
novel-craft context build "<scene-id>"
```

A scene card should answer want, resistance, turn, consequence, and causal link. If the scene repeats a prior function, add escalation, reversal, cost, reveal, payoff, or variation.
