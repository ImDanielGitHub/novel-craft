# Novel Craft Line Review

```bash
novel-craft lint line <draft.md>
novel-craft lint scene <draft.md>
novel-craft eval chapter <draft.md> --json
novel-craft revise <draft.md> --pass line
```

Filter words, passive voice, sentence fragments, and abstract emotion are leads, not automatic mistakes. Keep them when they serve POV, rhythm, withholding, helplessness, or voice.
