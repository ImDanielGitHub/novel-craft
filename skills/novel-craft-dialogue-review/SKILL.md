# Novel Craft Dialogue Review

```bash
novel-craft review <draft.md> --rubric dialogue
novel-craft eval chapter <draft.md> --json
novel-craft revise <draft.md> --pass dialogue
```

Ask what each speaker wants, who has leverage, what is dodged or withheld, and what relationship state changes by the end. Direct speech can be correct when earned; classify before cutting.
