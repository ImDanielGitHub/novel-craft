# Novel Craft Writing Support

Use this skill for names, docs, release notes, README cleanup, command explanations, and short project prose around Novel Craft or a novel project. For fiction drafting, use `novel-craft-agentic-writer` first.

## First Command

```bash
novel-craft writing guide
```

## Passes

1. Preserve meaning and boundaries.
2. Name things by what a busy person would ask for.
3. Put commands, output, and next steps before broad claims.
4. Replace vague confidence with proof, constraints, or honest judgement.
5. Cut generated-sounding filler.
6. Keep deterministic checks separate from human or LLM judgement.

## Gates

- Would a busy person understand the name in three seconds?
- Can an agent run the first command without guessing?
- Is the page useful before it tries to be impressive?
- Are claims evidenced, scoped, or clearly labelled as judgement?
