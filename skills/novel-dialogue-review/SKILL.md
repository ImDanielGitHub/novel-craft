# Novel Dialogue Review

Use this skill when a scene feels flat, talky, over-explanatory, or emotionally too direct.

## Workflow

```bash
novel-craft review <draft.md> --rubric dialogue
novel-craft analyse <draft.md>
```

Then revise only the dialogue pass:

```bash
novel-craft revise <draft.md> --pass dialogue
```

## Review Questions

- What does each speaker want?
- Who has leverage at the start?
- What is withheld, lied about, tested, bargained, or threatened?
- Does power shift by the end?
- Are characters stating motives and emotions too directly?
- Is exposition carried by conflict, asymmetry, or action?

## Rules

- Subtext is not obscurity. Keep reader comprehension while making the literal words do more than declare the meaning.
- Direct confession can be right when earned. Classify it before cutting it.
