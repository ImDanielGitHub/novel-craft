# Novel Craft Creativity Engine

Use this skill when a premise feels thin, too linear, too familiar, or not ready to draft.

## Workflow

```bash
novel-craft agent plan --idea "<seed idea>" --chapters 1 --genre <genre-or-profile> --json
novel-craft creative atlas --json
novel-craft creative brief --idea "<seed idea>" --genre <genre-or-profile> --must-include "<required fact>" --must-avoid "<forbidden claim>"
novel-craft creative tournament --idea "<seed idea>" --genre <genre-or-profile> --count 8 --json
```

## Contender Standard

Each contender should name:

- familiar genre appeal
- opening wound
- first hard choice
- freshness twist
- cost of advantage
- world pressure
- first-chapter image
- wider story engine
- page-turn reason

## Rules

- Widen before polishing.
- Mix ingredients causally instead of stacking labels.
- Make every power, system, status gain, romance turn, or reveal create a new cost.
- Usually dramatise macro appeal through micro-action first.
- Aim for the best novel every time; quality is not a special breakout mode.
